hệ thống được xây dựng tương thích với
- Xampp x64 7.4.3 VC15
- Phiên bản Php 7.4.3
- phiên bản Mysql 5.0.1
- Apache/2.4.41 (Win64) OpenSSL/1.1.1c

Tổng hợp cập nhật so với bản cập nhật 2.0.1
- Cập nhật FaISdb
- Cập nhật hệ thống điều hướng

- Phân chia thành 2 loại phiên bản
phiên bản sử dụng hệ thống Router (FaI-FwP-RhX-3.0.1)
phiên bản không sử dụng hệ thống Router (FaI-FwP-Pre-3.0.1)

Các tính năng

- Công cụ đọc và xử lý file json (FaIDbJ v1)

- Công cụ truy vấn và kiểm tra MySql theo mysqli (FaIDbSi v2)

- Hệ thống điều hướng router (FaIDr v3)

- Hệ thống Router (FaIRhX v3) xây dựng với htaccess của Xampp (chỉ tương thích cho xampp dành cho FaI-FwP-RhX-3.0.1)

- Công cụ trả header cho Resapi (FaIAhR v1)

- Công cụ kiểm tra thông tin url (FaIUrl v2)
